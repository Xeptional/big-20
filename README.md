# Birthday Website – Sarah's 20th

A sophisticated, interactive birthday page featuring a Memory Sprint game, friend messages, photo gallery, and more.

## Setup

1. Clone this repository.
2. Create a Supabase project and set up the `messages` table (see `sql/database.sql`).
3. Replace `SUPABASE_URL` and `SUPABASE_ANON_KEY` in `js/supabase.js`.
4. Add your own images to `assets/images/` and update the gallery paths in `index.html`.
5. Deploy to Netlify (drag-and-drop or git).

## Features

- Hero with animated title and photo
- Birthday message card
- Product review with animated bars
- Dynamic friend messages from Supabase
- Horizontal gallery with auto‑scroll and swipe
- Infinite Memory Sprint game (endless difficulty)
- Leave‑message page with spam protection
- Fully responsive and accessible

## Database Schema (Supabase)

```sql
CREATE TABLE messages (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  name TEXT NOT NULL,
  message TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
